Test Cases for Big-O Notation Estimator

This document provides a set of test cases to validate the accuracy of the Big-O Notation Estimator.

1. Constant Time Complexity: O(1)

function getFirstElement(arr) {
    return arr[0];
}

Expected Output: ✅ O(1) - Constant time

2. Linear Time Complexity: O(n)

function printElements(arr) {
    for (let i = 0; i < arr.length; i++) {
        console.log(arr[i]);
    }
}

Expected Output: ✅ O(n) - Linear time

3. Quadratic Time Complexity: O(n²)

function printPairs(arr) {
    for (let i = 0; i < arr.length; i++) {
        for (let j = 0; j < arr.length; j++) {
            console.log(arr[i], arr[j]);
        }
    }
}

Expected Output: ✅ O(n²) - Quadratic time

4. Logarithmic Time Complexity: O(log n)

function binarySearch(arr, target) {
    let left = 0, right = arr.length - 1;
    while (left <= right) {
        let mid = Math.floor((left + right) / 2);
        if (arr[mid] === target) return mid;
        else if (arr[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}

Expected Output: ✅ O(log n) - Logarithmic time

How to Test

1. Copy-paste the test case into the text area of your Big-O Notation Estimator.

2. Click on "Analyze Complexity".

3. Compare the output with the expected result.
